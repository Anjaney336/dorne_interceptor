# Results

Curated execution evidence for GitHub publication.

## Included In Git
- `results/graphs/`: representative figures from day-wise simulation/validation
- `results/metrics/`: key CSV outputs (benchmarks, mission summaries, sweeps)
- `results/reports/`: sovereign report figures, tables, and dossier PDF
- `results/samples/`: detection sample output images
- `results/manifest.json`: full local outputs vs published subset inventory

## Full Output Location (Local Only)
- `outputs/`

Raw videos and complete run logs remain local due size.
